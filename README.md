# 주문처리API

상점 등록 및 상품 등록 후 주문 처리를 하는 rest api 예제

### 기능

- 상점 목록조회
- 상점 상품목록 조회
- 상점 생성
- 상품 생성
- 상품 목록조회
- 상품 단일조회
- 주문 생성
- 주문 목록조회
- 주문 단일조회
- 주문 수정
- 주문 취소
- 주문 결제
